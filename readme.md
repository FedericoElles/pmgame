# PM the Game

[![dependencies](https://david-dm.org/FedericoElles/pmgame.png)](https://david-dm.org/FedericoElles/pmgame)

## Clone this project


## Installation

    npm install
    bower isntall


## Launch DB

    node lib/app.js
    
## Launch GUI

    grunt serve
   
   
Edit cards using the GUI. Once done, create a pull request to this repo to publish your creativity.
